# แอปคำนวณดอกเบี้ยทบต้น 10 ปี

วิธีใช้งาน
- เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์
- กรอกเงินตั้งต้น อัตราดอกเบี้ย (%) จำนวนครั้งที่ทบต่อปี และยอดเติมเงินรายเดือน (ถ้ามี)
- ปรับจำนวนปีตามต้องการ (ค่าเริ่มต้น 10 ปี)

สูตรที่ใช้
- มูลค่าในอนาคตจากเงินตั้งต้น: `FV_P = P * (1 + r/n)^(n*t)`
- เติมเงินรายเดือนแบบปลายงวด ใช้วิธีเทียบเท่าอัตราเติบโตต่อเดือน: `g_month = (1 + r/n)^(n/12)` และจำลองรายเดือน

หมายเหตุ
- การทบต้นนอมินัลปีละ `n` ครั้ง เมื่อมีการเติมรายเดือน จะคำนวณด้วยอัตราเติบโตเทียบเท่ารายเดือน `g_month` เพื่อความสอดคล้องของรอบเวลา
